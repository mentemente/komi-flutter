# core/theme

Esta carpeta contiene todo lo relacionado con el diseño visual global de la aplicación.

Define cómo se ve la app de forma consistente.

## ¿Qué va aquí?

- Colores de la app
- Tipografías
- Estilos de texto
- Tema claro / oscuro
- Configuración de ThemeData
- Espaciados y tamaños estándar

## Ejemplos de archivos

- `app_theme.dart` → Configuración principal del tema
- `app_colors.dart` → Paleta de colores
- `app_text_styles.dart` → Estilos de texto
- `app_spacing.dart` → Espaciados y tamaños comunes

## Reglas

- La UI debe consumir estilos desde aquí, no hardcodear colores o fuentes
- Mantener consistencia visual en toda la app
- Cambios de diseño global deben hacerse aquí
